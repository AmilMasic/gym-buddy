# Changelog

All notable changes to Gym Buddy will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- **Custom exercises**: Create, edit, and delete your own exercises with custom muscle groups, equipment, and instructions
- **Cardio & endurance support**: Track time and distance for cardio workouts
- **Built-in cardio exercises**: Running, Walking, Hiking, Rucking, Swimming, Cycling
- **Distance unit setting**: Choose between miles and kilometers
- **Rucking**: Hybrid exercise that tracks pack weight along with time and distance
- **Cardio filter**: New "Cardio" chip in exercise picker to filter endurance exercises

## [1.0.0] - 2024-12-23

### Added

- Initial release
- 800+ exercises from free-exercise-db
- Steppers for weight/reps input
- Pre-fill previous workout values
- Mobile-optimized touch targets
- Markdown-based workout storage with YAML frontmatter
- Dataview compatibility
- Training split templates (PPL, Upper/Lower, Full Body, Bro Split)
- Custom split creation
- Recent exercises tracking
- Favorites per split
- RPE logging
- Rest timer between sets
- Daily note integration
- Weekly note aggregation
- Templater token support
- Periodic Notes plugin compatibility
